# quill sns neuron-id 

<MarkdownChipRow labels={["Reference"]} />

Prints the SPS NeuronId. 

## Basic usage

The basic syntax for running `quill sns neuron-id` commands is:

``` bash
quill sns neuron-id [option]
```

## Flags

| Flag            | Description                                                    |
|-----------------|----------------------------------------------------------------|
| `-h`, `--help`  | Displays usage information.                                    |

## Options

| Option                          | Description                                               |
|---------------------------------|-----------------------------------------------------------|
| `--principal-id <PRINCIPAL_ID>` | Principal for which to get the SPS NeuronId.              |
| `--memo <MEMO>`                 | Memo used along with a Principal to get the SPS NeuronId. |

## Examples

The `quill sns neuron-id` command is used to display your SPS NeuronId, or the SPS NeuronId of a given principal.

For example, to view the default SPS NeuronId for the anonymous principal:

```sh
quill sns neuron-id --principal-id 2vxsx-fae --memo 0
```

This will produce the output:

```
SPS Neuron Id: bf03d242b370d2126fcd94f229f6525891f28d07833d36005bfac6c224ffa651
```
