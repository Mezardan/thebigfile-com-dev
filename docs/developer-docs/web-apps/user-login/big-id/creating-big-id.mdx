<MarkdownChipRow labels={["Intermediate", "Tutorial"]} />

## Overview

This guide walks through how to create an BIG ID. BIG ID is BIG's native form of identity. It is used by many dapps build on BIG for users to authenticate with.

## Create an BIG ID

To create an BIG ID, navigate to the II frontend URL: https://big-id.thebigfile.com/

Select 'Create New' from the UI.

Next, select 'Create Passkey'.
